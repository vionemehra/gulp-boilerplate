# Gulp Boilerplate

A simple boilerplate for build a static webpage in [gulp](https://gulpjs.com/).

## Getting Started 
### Installation

Install my-project with npm

```bash
  git clone https://github.com/vionemehra/gulp-boilerplate.git
  cd gulp-boilerplate
  npm install
```

### Development tasks

To run scss in development
```bash
  npm run css:dev
```
To run and watch page in developement
```bash
  npm run dev
```
To clean /dist folder
```bash
  npm run clean
```


## Deployment tasks
To deploy css run

```bash
  npm run css:build
```

To deploy this project run

```bash
  npm run build
```
